# SPARC TDD Mode

## Purpose
Test-driven development with TodoWrite planning and comprehensive testing.

## How to Invoke

In Claude Code, load this mode as a skill:
```
Skill("sparc:tdd")
```

## Core Capabilities
- Test-first development
- Red-green-refactor cycle
- Test suite design
- Coverage optimization
- Continuous testing

## TDD Workflow
1. Write failing tests
2. Implement minimum code
3. Make tests pass
4. Refactor code
5. Repeat cycle

## Testing Strategies
- Unit testing
- Integration testing
- End-to-end testing
- Performance testing
- Security testing
