# SPARC Workflow Manager Mode

## Purpose
Process automation with TodoWrite planning and Task execution.

## How to Invoke

In Claude Code, load this mode as a skill:
```
Skill("sparc:workflow-manager")
```

## Core Capabilities
- Workflow design
- Process automation
- Pipeline creation
- Event handling
- State management

## Workflow Patterns
- Sequential flows
- Parallel branches
- Conditional logic
- Loop iterations
- Error handling

## Automation Features
- Trigger management
- Task scheduling
- Progress tracking
- Result validation
- Rollback capability
